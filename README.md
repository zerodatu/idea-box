# idea-box

様々なアイデアを投稿する場所。もしかしたらアイデアを実現する可能性あり(実現可能なら解決に向けて頑張ってみます)

## このリポジトリの使い方

みんなが困ってることや、これがこうできたらいいのになぁみたいなことを投稿するだけ<br>
issueでもmdファイルでもいいよ

## どうしてこれを作ったのか

いろんな困りごとはアイデアになるし、それが解決できるシステムなどができれば皆が喜ぶと思ったから

## 実現できないものはどうする？

今は実現できないねってことでオミット扱いにします<br>
でもなんとなく実現できそうなものは手を付けていきます
