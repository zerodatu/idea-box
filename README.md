# idea-box

様々なアイデアを投稿する場所。もしかしたらアイデアを実現する可能性あり(実現可能なら解決に向けて頑張ってみます)

## このリポジトリの使い方

みんなが困ってることや、これがこうできたらいいのになぁみたいなことを投稿するだけ<br>
issueで投げてもいいし、プルリクエストで出してもらっても構いません。<br>
内容としてこれは改善すると良いと判断したものはどんどんテーブルに入れます。

## どうしてこれを作ったのか

- いろんな困りごとはアイデアになるし、それが解決できるシステムなどができれば皆が喜ぶと思った
- 自分の開発スキルで社会を良くしたいという気持ち
- 楽曲制作でも場が明るくできると思うので、そういうアイデアも実現できたらいいなと思っている。

## 実現できないものはどうする？

今は実現できないねってことでオミット扱いにします<br>
でもなんとなく実現できそうなものは手を付けていきます
